**H**yper**T**ext **T**ransfer **P**rotocol

[[하이퍼텍스트]]를 빠르게 교환하기 위한 프로토콜
요청과 응답으로 구성되어 있으며 80번 [[포트]]를 주소 사용한다.

- Client-Server 구조 : 클라이언트가 요청하면 서버가 응답
- 무상태성 : 서버가 클라이언트의 상태를 저장하지 않음
- 비연결성 : 응답을 받으면 바로 연결을 끊음

**포함되는 정보.**
- HTTP 버전 유형
- URL
- HTTP 메서드
- HTTP 요청 헤더
- HTTP 본문 (Optional)
