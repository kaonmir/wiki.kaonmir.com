Authentication factor, [[인증]] 시스템이 사용자의 신원을 확인하기 위해 확인하는 요소

**종류**
1. 사용자만 '**아는 것**'
    - 아이디, 비밀번호
    - PIN 코드
    - 보안 질문
    - API 키
2. 사용자만 '**가진 것**'
    - 소프트 토큰 : 임시 인증 번호(이메일, 문자 등)
    - 하드 토큰 : USB, 블루투스 장치. [[OTP]]
3. 사용자만 '**있는 것**'
    - 얼굴, 지문, 목소리, 서명
4. 추가 인증 요소
    - 위치, 시간

[[HTTPS]]는 웹 사이트의 [[SSL 인증서]]를 사용하여 웹 사이트가 인증한다.
