요즘 공유기는 스위치, 게이트웨이, 라우터 기능이 모두 결합된 장치입니다.

| 장비    | 설명                                      |
| ----- | --------------------------------------- |
| 게이트웨이 | 서로 다른 네트워크(사설망, 공개망, 전화망, …) 간에 데이터를 변환 |
| 스위치   | OSI 2 계층에서 MAC 주소를 읽고 어디로 데이터를 보낼지 결정   |
| 라우터   | OSI 3 계층에서 IP 주소를 읽고 어디로 데이터를 보낼지 결정    |
