**커널은 운영체제의 한 부분**이다
 - 커널 = 하드웨어↔소프트웨어
- 운영체제 = 하드웨어↔사용자

예시를 들자면,

- 커널 = 엔진
- 운영체제 = 자동차
- 배포판 = 자동차 모델(제네시스, 토요타)

| 왼쪽                                  | 오른쪽                                       |
| ------------------------------------- | -------------------------------------------- |
| 자동차를 탄다                         | 운영체제를 쓴다                              |
| 제네시스를 탄다                       | 우분투를 쓴다                                |
| 제네시스는 자동차이다                 | 우분투는 운영체제이다                        |
| 현대에서 새 제네시스 모델을 출시했다. | Canonical에서 새 우분투 배포판을 릴리즈했다. |
